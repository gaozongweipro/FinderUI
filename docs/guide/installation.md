# 安装指南

## 环境要求

- Node.js >= 14.0.0
- Vue >= 3.0.0
- Element Plus >= 2.0.0

## 安装

### 使用包管理器

推荐使用 npm 或 yarn 安装：

```bash
npm install @gaozongweipro/find-ui
```

安装后在main.js中引入

```js
import FindUI from "@gaozongweipro/find-ui";
import "@gaozongweipro/find-ui/dist/find-ui.css";

app.use(FindUI);
```
